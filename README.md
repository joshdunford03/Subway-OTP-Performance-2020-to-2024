# NYC Subway On-Time Performance Analysis (2020-2024)

This project analyzes how subway On Time Performance varies by line and day type using pandas and data visualization.

## 📊 Key Findings
*   **Top Performers:** The **7, L, and G lines** consistently had the highest OTP (On-Time Performance) because they use **modern signaling** and dedicated tracks.
*   **Reliability Gaps:** Longer routes like the **2 train** showed significant delays, hitting a low of **31% OTP** in February 2024 due to its length and frequent stops.
*   **The "Pandemic Effect":** All lines performed better in 2020 due to low ridership, which allowed trains to leave stations faster.
